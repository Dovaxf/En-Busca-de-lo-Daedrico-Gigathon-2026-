# EN BUSCA DE LO DAÉDRICO

## Requisitos

* Python 3.11
* Biblioteca estándar de Python

## Ejecución

Ejecutar:

main.py

## Descripción

Simulador de exploración ambientado en Skyrim.

El jugador controla al Dovahkiin y debe encontrar la legendaria Navaja de Mehrunes mientras gestiona su energía y explora un mundo bidimensional.

La ruta recorrida se representa gráficamente mediante Turtle durante la expedición.

## Controles

1. Mover al norte
2. Mover al sur
3. Mover al este
4. Mover al oeste
5. Descansar
6. Explorar
